# 🎓 University Campus Area Network Project

## 📌 Overview

This project simulates a **University Campus Area Network** using Cisco Packet Tracer.
It demonstrates **network segmentation, inter-VLAN communication, and service integration** in a scalable and secure environment.

---

## 🚀 Features

✨ VLAN-based network segmentation
🔄 Dynamic routing using RIPv2
⚡ Automatic IP assignment using DHCP
🌐 DNS server for domain resolution
🖥️ Web server for hosting university website
📧 Email server for internal communication

---

## 🏗️ Network Design

The network is designed using a hierarchical approach:

* 🔹 **Access Layer** – Connects end devices (PCs, laptops)
* 🔹 **Routing Layer** – Handles inter-VLAN routing using RIPv2
* 🔹 **Servers Layer** – Provides essential services
